Thank you for using HappyWhale application!

----------------------------------------------------------

This model architecture:
1. YOLOv5 	- pretrained by us to detect whale and dolphin body parts
2. ResNet50 - DCNN to obtain the whale or dolhin's individual ID

----------------------------------------------------------
Prequisitions:
1. Open Spyder (Python 3.9) 5.1.5 IDE
Inside Spider's terminal run the following line:
pip install -r yolov5/requirements.txt

How to run the HappyWhale application:
1. Open HappyWhaleApp.py in Spyder (Python 3.9) 5.1.5 IDE
2. Click run
3. Click "Upload" and load a whale or dolphin image
4. Click "Detect" to acquire the whale or dolphin's individual ID
5. The individual ID will appear on screen
